# Changelog

All notable changes to NotebookAwesome are documented here.

## [0.1.0] — 2026-07-21

### Added
- Cell number label on every Jupyter notebook cell (auto-updates)
- Renameable cell headers stored in `.ipynb` cell metadata
- Navigation sidebar panel with cell list, type badges, and click-to-navigate
- Rename cell command via status bar, toolbar menu, command palette, and nav panel
- Clear cell name command
- Cross-fork support (VS Code, Cursor, Windsurf, and other VS Code-compatible editors)
